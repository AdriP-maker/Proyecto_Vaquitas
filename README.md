# Proyecto Vaquitas - HOFLOC S.A.

Plataforma Web de Gestión Agropecuaria para la Finca HOFLOC S.A.

Este repositorio contiene la etapa inicial del análisis y arquitectura de la plataforma dividida para tres equipos de desarrollo: UI/UX, Base de Datos y Lógica (Formularios).

## Estructura de Documentación

Los documentos iniciales de análisis arquitectónico y responsabilidades están dentro de la carpeta `docs/`.

- `01_Mapa_General_del_Sistema.md` - Visión y alcance general de los módulos (Público e Interno).
- `02_Asignacion_Responsabilidades.md` - Responsabilidades divididas y dependencias cruzadas entre equipos.
- `03_Reglas_de_Negocio.md` - Lista de reglas operativas basadas en datos reales.
- `04_Esquema_Base_de_Datos_ERD.md` - Esquema de las entidades y relaciones.
- `05_Flujos_Datos_Criticos.md` - Descripción imperativa de flujos esenciales.
- `06_Plan_Desarrollo_Recomendado.md` - Plan iterativo y fases.
- `07_Roles_Requisitos_Escenarios.md` - Roles del sistema, 37 requisitos funcionales, 6 escenarios de uso y 25 requisitos no funcionales.
